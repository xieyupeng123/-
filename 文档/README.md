# 西游记·观音院 - Pygame游戏项目

## 项目概述
基于Pygame和Tiled地图的RPG游戏，讲述西游记中"祸起观音院"的故事。

## 项目结构（面向对象，多文件结构）

```
软件项目实训/
├── main.py                    # 游戏主程序入口
├── test_game.py               # 测试程序
├── README.md                  # 项目说明
│
├── map/                       # 地图模块
│   ├── __init__.py
│   └── tmx_map.py             # Tiled地图类
│
├── sprites/                   # 精灵模块
│   ├── __init__.py
│   ├── sprite.py              # 精灵基类
│   ├── animated_sprite.py     # 动画精灵类
│   ├── player.py              # 玩家类
│   └── npc.py                 # NPC类
│
├── scenes/                    # 场景模块
│   ├── __init__.py
│   ├── scene.py               # 场景基类
│   └── village_scene.py       # 村庄场景
│
├── resource/                  # 资源文件
│   ├── img/                   # 图像资源
│   │   ├── village.jpg        # 村庄背景图
│   │   ├── swk/               # 孙悟空精灵（4方向）
│   │   ├── god/               # 土地公精灵
│   │   └── elder/             # 长老精灵
│   │
│   ├── tmx/                   # Tiled地图文件
│   │   ├── village.tmx        # 村庄地图
│   │   └── temple.tmx         # 寺庙地图
│   │
│   └── font/                  # 字体文件
│
└── 视频/                      # 教学视频
```

## 已完成的功能

### 1. Tiled地图加载 ✓
- 使用pytmx库加载TMX格式地图
- 支持多图层渲染（图像层、瓦片层、对象层）
- 地图大小: 3780x2395 像素
- 支持障碍物检测

### 2. Sprite应用 ✓
- 精灵基类（Sprite）
- 动画精灵类（AnimatedSprite）- 支持4方向动画
- 玩家类（Player）- 键盘控制移动
- NPC类（NPC）- 自主移动

### 3. 面向对象结构 ✓
- 多文件、多模块结构
- 类的继承和封装
- 模块化设计

## 运行方法

### 测试程序
```bash
.venv/Scripts/python test_game.py
```

### 完整游戏
```bash
.venv/Scripts/python main.py
```

## 操作说明
- **方向键**: 移动玩家（孙悟空）
- **ESC**: 退出游戏

## 技术要点

### 1. 地图层次结构
```
┌─────────────────────────────┐
│      精灵层（玩家、NPC）      │  ← 最上层
├─────────────────────────────┤
│      对象层（障碍物位置）      │
├─────────────────────────────┤
│      图块层（建筑物等）       │
├─────────────────────────────┤
│      图像层（背景图片）       │  ← 最底层
└─────────────────────────────┘
```

### 2. 视窗系统
- 地图大小 > 视窗大小（3780x2395 > 800x600）
- 视窗跟随玩家移动
- 边界检测防止越界

### 3. 精灵系统
```
Sprite（基类）
├── AnimatedSprite（动画精灵）
│   ├── Player（玩家）
│   └── NPC（非玩家角色）
```

### 4. 场景系统
```
Scene（基类）
└── VillageScene（村庄场景）
```
